Download SoundCloud music at 128kbps with album art and tags

Tested on Linux (Manjaro)

Installation

# clone the repo $ git clone https://github.com/sdushantha/soundcloud-dl.git  # change directory to soundcloud-dl $ cd soundcloud-dl  # install the requirements $ pip3 install -r requirements.txt

Usage

usage: soundcloud-dl.py [-h] url

Example

python3 soundcloud-dl.py https://soundcloud.com/marshmellomusic/friends
